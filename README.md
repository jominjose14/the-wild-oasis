# The Wild Oasis

A hotel management web app - [Link](https://the-wild-oasis-mgmt.vercel.app/)
<br><br>

**Features**

- Stats dashboard with KPIs
- CRUD (users, cabins, bookings)
- JWT Authentication
- Dark Mode

<br>

**Tech Stack**

- React
- React Router
- Tanstack Query
- React Hook Form
- Recharts
- Styled Components
- Supabase
